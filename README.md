   ・pwd
  現在のディレクトリのパスを表示する
  ・ls
  現在のディレクトリにあるファイル、ディレクトリを表示する
  ・cd <path>
  change directory でディレクトリのパスを変更する

  基本的には
  pwd で現在どこにいるか把握する
  cd でどうやったら辿り着けるか考える
  cd で移動しつつ、適宜 ls で確認する
  cd .. # .. を利用すると一つ上の階層に行く
  cd hoge # hoge ディレクトリに移動する

  fn＋F12　webで色々表示ー情報を4oに入れると良い
